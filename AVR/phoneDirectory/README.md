# Phone Directory, 전화번호부
Projects In the Course, Microprocessor application.

마이크로프로세서 응용에서 진행했던 AVR을 이용한 전화번호부 프로젝트이다.

기본적으로 3X4의 키버튼과 2*4의 키버튼 두 개를 사용하며, 다음의 기능을 갖는다.

* Home : 전화번호 목록 보기,
* Insert : 전화번호 추가
* Search : 전화번호 검색
* Delete : 전화번호 삭제

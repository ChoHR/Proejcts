# Recognition by Histogram
Projects In the Hanyang university.

히스토그램을 이용하여 물체를 찾는 프로그램 소스코드이다.

필요한 사진은 두 개이다. 
* 찾고자하는 물체만 있는 사진
* 찾고자하는 물체가 있는 사진

OpenCV 2.4.10 버전을 사용했다. 다른 버전에서 실행하기 위해서는 Used for library linking이라고 써있는 주석 밑에 있는 llb 이름들을 버전에 맞게 수정해야한다.
